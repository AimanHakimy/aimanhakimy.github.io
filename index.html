🔍 Section 1: Career Exploration (Understanding the Field)

1. Selected Role: Software Developer (Security-focused)

Positioning application engineering workflows at the front line of software perimeter and internal code layer defenses.

Job Responsibilities:

  • Reviewing application source code to identify and mitigate software bugs and flaws.
  • Integrating automated security tools directly into the development lifecycle.
  • Ensuring web and mobile digital platforms protect user identity through proper validation protocols.
  • Documenting system architecture vulnerabilities to optimize platform integrity.

Required Skills:

  • Understanding secure web frameworks and modern development environments.
  • Core competency in web technologies, routing systems, and input filtering logic.
  • Practical understanding of security frameworks, privacy-by-design principles, and basic web vulnerability concepts.

Relevance to Modern Cybersecurity:

Digital media platforms continuously collect, display, and distribute information. Because modern systems face persistent security threats at the application layer, media technologists need a basic awareness of authentication, data privacy, safe communication, responsible content sharing, and secure web practices to ensure that deployments remain safe for end-users.
